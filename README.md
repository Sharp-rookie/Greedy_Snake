### 贪吃蛇1.0

##### 规则

1. 初始蛇身长为2，向右移动，每吃到一次食物身长加1，积分加10
2. 键盘wasd控制移动方向
3. 碰到墙壁、障碍物或者吃到自身时死亡，游戏结束

&nbsp;

##### 运行方法

1. 打开命令行窗口，输入 `gcc -o greedy_snake.c greedy` 进行编译

2. 输入 `./greedy_snake` 开始运行
